plp_ds(parser,file)

An object that extracts a probabilistic logic program in the distribution semantics from a parsed probabilistic logic program.

  • parser – A parser implementing parserp.
  • file – A file descriptor (an atom) containing a probabilistic logic program to be parsed by the parser.
