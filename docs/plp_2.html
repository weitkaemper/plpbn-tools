object

plp(parser,file)

An object that extracts an internal probabilistic logic program representation and a database from a parsed probabilistic logic program.

  • parser – A parser implementing parserp.
  • file – A file descriptor (an atom) containing a probabilistic logic program to be parsed by the parser.
